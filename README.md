# Simple blog app

## Get started

### For launching project u have to:

- Open client folder
  - open console
  - ```bash
    $ npm install
    ```
  - ```bash
    $ npm start
    ```
- Open server folder
  - open console
  - ```bash
    $ npm install
    ```
    - ```bash
      $ npm run start
      ```

#### Ports 3000 and 4000 must to be free

## Stack of the app

React, typescript, redux-toolkit, axios, nest.

Simple REST API.
